Siberian Ehr Framework

Project maintained by ilizunov

What is SiberianEhr?

SiberianEhr is a set of full-featured UI components for openEhr-based medical systems.

Developers

SiberianEhr is being developed by netEagles company from Russia.

Licencing

SiberianEhr can be used free both for commercial and non-commercial projects.

Support

Demonstration, documentation and API reference available for this project.
